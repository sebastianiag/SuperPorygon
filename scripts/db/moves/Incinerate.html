Effects

Incinerate deals damage and if the target is holding a Berry, it is burnt and becomes no longer usable.

If Incinerate would be super-effective and the target is holding an Occa Berry, the Berry activates before the move hits so does not get burnt.

Changes

In Generation 5, Incinerate has base power 30.
