Effects

Echoed Voice deals damage starting at base power 40, and increases by 40 each turn if used by any PokÃ©mon on the field, up to a maximum base power of 200.

It does not reset if it misses, as long as the move is attempted each turn. The power does not increase during a turn (if multiple PokÃ©mon use it), only after each turn.
