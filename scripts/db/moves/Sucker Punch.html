Effects

Sucker Punch deals damage and has a priority of +1. However, the move only works if the target uses a direct attack in the same turn. If the target is not preparing an attack, Sucker Punch will fail.

Moves with a higher priority always go first, regardless of a PokÃ©mon's speed. If two moves with the same priority are used, the faster PokÃ©mon goes first as usual.
